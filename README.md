# Demand Forecasting for Retail Inventory Management

## Project Overview

This project was completed as part of the Transvolt Data Science Internship Assessment.

The objective is to forecast weekly product demand across multiple supermarkets using historical sales and promotional information.

## Dataset

- demand.csv
- promotions.csv

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- LightGBM
- Scikit-learn

## Workflow

- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Walk-Forward Validation
- LightGBM Forecasting
- Model Evaluation
- Business Insights

## Results

- MAE : 11.99
- RMSE : 18.49
- R² : 0.9787
- MAPE : 2.21%
- WAPE : 2.27%

## Business Outcome

The forecasting model helps improve inventory planning by reducing stock-outs and minimizing excess inventory.
